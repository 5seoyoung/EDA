## 성과
3위

## 개요
본 프로젝트는 AI 및 빅데이터 분석 학회 D&A의 Basic Session EDA Competition에서 수행한 서울시 범죄 데이터 분석입니다. 본 분석에서는 범죄 유형에 따른 인프라 투자 우선순위를 파악하여 서울시의 안전성을 개선하기 위한 인사이트를 도출하는 것을 목표로 하였습니다.
